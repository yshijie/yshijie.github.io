Android Studio常用快捷键

发表于 2017-06-08 | 分类于 Android Studio

Android Studio快捷键

整理一些关于Android Studio中的常用快捷键，为方便说明，加上与Eclipse中的对比。
  • Ctrl + Y ：删除光标所在的一行（对应Eclipse中的Ctrl + D，若选择多行，也可以全部删除，只需要选中某行的一部分内容即可）

  • Ctrl + D ：将光标所在行的内容复制，并粘贴到该行的下一行

  • Ctrl + O ：重写父类的方法，使用Ctrl + O可以直接打开父类所有可重载的方法列表，选择对其进行重写。

  • Ctrl + P ：查看该方法的参数
  • Ctrl + U ：定位到父类，（光标在一个继承父类重写的方法上）

  • Ctrl + E ：最近访问，查看最近访问的文件列表

  • Ctrl + Tab ：快速切换打开的文件。

  • Alt + J ：选择多出文本（例如我们需要修改代码中全部的一个变量名mTextView，我们将第一个mTextView选中，之后一直按Alt + J，就可以挨个将后面其他的mTextView选中，一起修改）
  • Alt + F7 ：查找该方法在何处被调用
  • Alt + Enter ：快速矫正代码，提示错误代码的修改，类型转换等（对应于Eclipse中的Ctrl + F1）

  • Alt + Up/Down ：在方法和内部类之间跳转

  • Alt + 关闭：在代码编辑区域打开了好多文件，需要关闭其他不需要的，通常做法是在需要留下的文件上面右键->Close Others，其实只需要按住Alt键，点击要留下的文件的关闭按钮，就会实现同样的效果。

  • Shift + F4 ：将当前打开的文件，另起一个窗口来进行显示。
  • Ctrl + Shift + J ：合并行（将多行代码合并成一行，合并多行注释；合并两行代码，例如 int a;a=1; –> int a=1; 合并多行字符串等）
  • Ctrl + Shift + Enter ：补全语句（可以将该行的末尾补全封号；if、while、for等补全后面的圆括号和花括号；方法后面补全花括号）
  • Ctrl + Shift + “+”/“-“ ：折叠/展开代码
  • Ctrl + Shift + F12 ：隐藏所有面板
  • Ctrl + Shift + I ：在当前界面查看一个方法或者类的具体实现
  • Ctrl + Shift + Up/Down ：上下移动方法
  • Ctrl + Alt + L ：格式化代码，即将代码按层次缩进（对应于Eclipse中的Ctrl + Shift + F，该快捷键会与QQ的锁定快捷键冲突，需要将QQ的这个热键关闭或修改）
  • Ctrl + Alt + Space ：代码提示（对应于Eclipse中的Alt + /）
  • Ctrl + Alt + M ：提取方法

  • Alt + Shift + Up/Down ：上下移动光标所在行（对应于Eclipse中的Alt + 上下键）

  • Alt + Shift + Left/Right：在编写xml布局时，可以在Design和Text之间直接切换，省得用鼠标点

书签
  • F11 ：添加/移除书签
  • Ctrl + F11 ：添加/移除书签（带标记）
  • Shift + F11 ：显示全部书签
与剪切板对比
  • 选中内容，右键，选择Compare With Clipboard
使用Enter和Tab进行代码不全
  • Tab ：从光标插入处补全代码，并删除后面的代码，直到遇到点号、圆括号、分号或有空格为止
  • Enter ：从光标插入处不全代码，对原来的代码不做任何操作
